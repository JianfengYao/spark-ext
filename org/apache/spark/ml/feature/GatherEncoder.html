class GatherEncoder extends Estimator[GatherEncoderModel] with GatherEncoderParams

Encode categorical values collected by Gather transformation as feature vector using dummy variables inside AttributeGroup with attached metadata

cookie_id | sites
----------|------------------------------------------------------------------------
cookieAA  | [{ site_id: 1, impressions: 15.0 }, { site_id: 2, impressions: 20.0 }]
cookieBB  | [{ site_id: 2, impressions: 7.0 }, { site_id: 3, impressions: 5.0 }]

transformed into

cookie_id | site_features
----------|------------------------
cookieAA  | [ 15.0 , 20.0 , 0   ]
cookieBB  | [ 0.0  ,  7.0 , 5.0 ]

Optionally apply dimensionality reduction using top transformation:

  • Top coverage, is selecting categorical values by computing the count of distinct users for each value, sorting the values in descending order by the count of users, and choosing the top values from the resulting list such that the sum of the distinct user counts over these values covers c percent of all users, for example, selecting top geographic locations covering 99% of users.
