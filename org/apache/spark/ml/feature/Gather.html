org.apache.spark.ml.feature

Gather

class Gather extends Transformer with GatherParams

Inspired by R tidyr and reshape2 packages. Convert long org.apache.spark.sql.DataFrame with values for each category into wide org.apache.spark.sql.DataFrame, applying aggregation function if single category has multiple values

cookie_id | site_id | impressions
----------|---------|--------------
 cookieAA |   123   | 10
 cookieAA |   123   | 5
 cookieAA |   456   | 20

gathered using sum agregate

cookie_id | output_col
----------|------------------------
cookieAA  | [{ site_id: 123, impressions: 15.0 }, { site_id: 456, impressions: 20.0 }]
