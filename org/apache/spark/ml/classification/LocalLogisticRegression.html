org.apache.spark.ml.classification

LocalLogisticRegression

class LocalLogisticRegression extends ProbabilisticClassifier[Vector, LocalLogisticRegression, LogisticRegressionModel] with LogisticRegressionParams with Logging

Local version of LogisticRegression When DataFrame is too small that it can easily fit into single node it doesn't make sense to build model using RDD, it can be built on single node. Essentially using Spark as distributed Executor
